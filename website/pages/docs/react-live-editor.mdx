# React Live Editor

Build interactive documentation using react live code editor.

<carbon-ad />

## Write live code

To include a live code editor, add `live` in the head of your code block. It allows to write a single JSX expression:

````
```js live
<div>Example</div>
```
````

**Result:**

```js live
<div>Example</div>
```

## Write multi-expression examples

By default only a single JSX expression is allowed. It is common to have to write more complex examples. The `noInline` flag makes it possible. When activated, you have to call a `render()` method to display example:

````
```js live noInline
render(<div>Example</div>)
```
````

**Result:**

```js live noInline
render(<div>Example</div>)
```

> To know more about `noInline` mode, read [react-live documentation](https://github.com/FormidableLabs/react-live#what-code-can-i-use).

## Import modules

Importing modules is possible in a React Live Editor using ESM `import` syntax. **It is an emulation of the import, it means you don't have access to all modules by default.** To be able to impot a module, first you have to specify it using `LiveConfig` component.

```
---
title: Doc Page
---

import { Box } from '@xstyled/styled-components'
import { LiveConfig } from 'smooth-doc/components'

<!-- Define `@xstyled/styled-components` module as `{ Box }` -->
<LiveConfig modules={{ '@xstyled/styled-components': { Box } }} />
```

Once defined, you have access to the module in live editor:

```js live noInline
import { Box } from '@xstyled/styled-components'

// I can use lodash here.
render(<Box color="red">Example</Box>)
```
